# 🍜 味觉冒险家

一个为大学生打造的智能美食推荐平台，结合AI技术和趣味互动，让选择美食变成一场有趣的冒险！

## ✨ 核心功能

- 🧠 **AI智能推荐**: 基于MBTI性格、星座特征、传统黄历的个性化推荐
- 🎰 **趣味互动**: 转盘推荐、盲盒惊喜、性格测试等游戏化体验
- 💬 **美食社区**: 校园美食分享平台，发现同学推荐的隐藏美食
- 🗺️ **地图探索**: 交互式地图展示周边餐厅位置和详情
- 📱 **响应式设计**: 完美适配各种设备

## 🎯 创新亮点

- 个性化推荐算法结合心理学和传统文化
- 游戏化交互设计让选择美食充满惊喜
- 专为大学生打造的校园社区生态
- 现代化的用户界面和流畅的交互体验

## 🚀 技术栈

- 前端：HTML5 + CSS3 + JavaScript
- 设计：响应式布局 + 动画效果
- 特色：纯前端实现，无需后端服务

## 📊 平台数据

- 8+ 精选餐厅
- 4 所合作高校
- 16 种MBTI类型推荐
- 12 个星座推荐系统
- 100+ 社区动态
